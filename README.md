## Simple implementation of golang channels
[![License](https://img.shields.io/badge/License-BSD_2--Clause-orange.svg)](https://opensource.org/licenses/BSD-2-Clause)
[![CMake Actions Status](https://github.com/rustpgrus/channels/workflows/CMake/badge.svg)](https://github.com/rustpgrus/CMake/actions)

### Build project

Execute commands in terminal: `make build`.

### Build tests

Execute command in terminal: `make build-tests`.

### Run program and tests

Execute command in terminal:

1. Program: `make run-prog`.
2. Tests: `make run-tests`.
